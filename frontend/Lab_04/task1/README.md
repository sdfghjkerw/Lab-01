# Лабораторная работа 3.1 — React Components and State

## Описание
В лабораторной работе реализованы задания, направленные на освоение работы с состоянием компонентов в React с использованием хука `useState`.  
Проект создан с использованием React 18+ и Vite.

## Task 1: Счётчик с инкрементом и декрементом
Цель: создать компонент, который управляет числовым состоянием.

Реализация:
- Используется `useState` для хранения значения счётчика.
- Добавлены кнопки:
  - "Increment" — увеличивает значение на 1.
  - "Decrement" — уменьшает значение на 1.
- Отображение текущего значения счётчика.

## Task 2: Счётчик с шагом и историей
Цель: расширить функционал счётчика.

Реализация:
- Добавлен проп `step`, который задаёт величину изменения счётчика.
- Используется массив `history` для хранения всех значений после операций.
- Добавлен счётчик операций `operationCount`.
- Кнопки:
  - "Increment" — увеличивает значение на `step`.
  - "Decrement" — уменьшает значение на `step`.
  - "Reset" — сбрасывает состояние к начальному значению, очищает историю и количество операций.

## Task 3: Управление несколькими состояниями
Цель: закрепить работу с несколькими состояниями в одном компоненте.

Реализация:
- Используются несколько вызовов `useState` для разных частей состояния:
  - `count` — текущее значение.
  - `history` — массив значений.
  - `operationCount` — количество операций.
- Каждое действие обновляет все связанные состояния.
- Компонент отображает:
  - Текущее значение счётчика.
  - Историю изменений.
  - Количество операций.

## Технические требования
- React 18+, функциональные компоненты и хуки.
- Обработка всех состояний: инкремент, декремент, сброс.
- Использование иммутабельных обновлений состояния (`setState` с новым значением или копией массива).
- Код без ошибок, с понятными именами и комментариями.
