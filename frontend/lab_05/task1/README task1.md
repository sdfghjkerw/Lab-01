# Lab 5 - Task 1: TypeScript Component & Props

## Описание проекта

Этот проект реализует компонент UserCard, демонстрирующий использование TypeScript для типизации пропсов (props) и работу с children в React.

## Основные концепции TypeScript

### 1. Типизация пропсов (Props Interface)

Вместо использования any мы определяем интерфейс UserCardProps. Это обеспечивает безопасность типов, гарантируя, что компонент получает данные правильной структуры.

Пример интерфейса:

```typescript
interface UserCardProps {
    user: User;
    isActive: boolean;
    children?: React.ReactNode;
}

```

### 2. Типизация children

Тип React.ReactNode используется для типизации children, что позволяет передавать в компонент любые допустимые в React элементы (текст, другие компоненты, HTML-теги).

### 3. Типизация функционального компонента

Мы используем тип React.FC<UserCardProps> (или просто указываем тип пропсов в аргументах функции), чтобы TypeScript проверял правильность использования пропсов внутри компонента.
